extract_fields/1

extract_fields(BinaryMessage::binary()) -> list(integer())

Extracts a list of field IDs from a binary representation of an ISO 8583 message. The result is returned as a 2-tuple: a list of field IDs and the remainder of the message excluding the bit map.
